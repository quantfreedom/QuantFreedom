    Add Token To Script¶

    • go to the github page here https://github.com/QuantFreedom1022/live_bots/blob/main/aws_script.sh
    • then on the right side click the pencil edit icon
    • on line 7 change where is says %%%%%%%GITHUB_TOKEN with yoru github token
    • on line 13 replace %%%%%%%STRAT_FOLDER with the name of the folder of the strat you want to run ... in this case it would be 02_rsi_rising_falling
    • on line 14 replace %%%%%%%RUN_FILE.py with the main.py in the folder of the strat you want to run ... in this case it will be rsi_main.py
    • now replace the %%%%%%%API_KEY with your api key
    • replace the %%%%%%%SECRET_KEY with your secret key
    • now replace %%%%%%%USE_TEST_NET with a true if you want to use the testnet and a false if you want to use the main net ... all lower case
    • finally replace %%%%%%%%EXCHANGE_NAME with the name of the exchange you want to trade on ... in this case it is mufex or bybit ... all lower case
